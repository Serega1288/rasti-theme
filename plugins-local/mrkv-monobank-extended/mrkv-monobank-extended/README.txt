== Description ==

Плагін додає на ваш сайт спосіб оплати через еквайринг Монобанку (прийом банківських карток).

Раніше плагін також підтримував Mono Checkout (швидка оплата поза стандартним чекаутом WooCommerce), однак цей функціонал було вилучено через припинення підтримки з боку API Монобанку.

* Спосіб оплати Еквайринг

* Можливість вибрати статус замовлення для успішних оплат
* можливість вибрати логотип моно для білого та темного фону

= 👉️ Як підключити еквайринг =

1. Відкрийте рахунок ФОП в Монобанк
2. Зайдіть в [веб кабінет] (https://web.monobank.ua/) та підключіть онлайн-оплату для сайту (буде покроковий майстер підключення)
3. Вам запропонує встановити офіційний плагін від моно, ввічливо відмвляємося, і встановлюємо плагін від morkva
4. Система згенерує для вас токен АРІ, який потрібно ввести в налаштування плагіна
5. Це ще не все - ваш магазин повинні активувати у сліубі підтримки Моно, переконайтеся, що на сайті є логотип Plata by mono, сторінка з умовами оплати і доставки, контактні дані і можливо ще якісь вимоги від банку. Моно досить лояльні, якщо швидко все зробити - активують миттєво.
6. Все, можете приймати оплати

= 👉️ Режим тестування =

В плагіні є два режими для тестування: глобально для всіх, та лише для залогованих адміністраторів. В другому випадку ви можете тестувати оплати окремо від покупців.

Отримати тестовий ключ АРІ [можна тут.] (https://api.monobank.ua)

= 👉️ Як працює повернення на сторінку подяки =

Плагін повертає на сторінку подяки (thank you page), але є нюанс. Плагін передає значення redirect_url при кожному запиті, але по замовчуванню АРІ його не опрацьовує. Тому, після налаштування плагіна і активації вам еквайрингу службою підтримки банку, попросіть їх активувати перенаправлення покупців на сторінку подяки на вашому сайті.

Потрібна підтримка чи додатковий функціонал? support@morkva.co.ua
